## Project Title

LSST Data Pipeline for Rubin Observatory and DeepLense

## Overview

The proposal focuses on building a reusable data pipeline that connects Rubin Observatory LSST data access tools with DeepLense machine learning workflows. The goal is to make it easier to retrieve, preprocess, and organize astronomical survey data for ML tasks such as lens finding, super-resolution, and related classification problems.

## Main Objectives
```
Build a Butler-based LSST data retrieval pipeline
Create preprocessing utilities for astronomical image cutouts
Prepare ML-ready datasets for DeepLense workflows
Support integration with lensing-related machine learning tasks
Provide documentation, testing, and reproducible examples
```
